# ZIMA App

ZIMA es una aplicación móvil desarrollada con React Native y Expo diseñada para el monitoreo metabólico personal.

La aplicación permite registrar biometría corporal, analizar información nutricional y generar recomendaciones basadas en datos metabólicos.

## Características principales

* Registro biométrico de cetonas
* Registro de peso corporal
* Escáner de tablas nutricionales
* Cálculo automático de carbohidratos netos
* Motor metabólico SIM
* Avatar evolutivo basado en progreso
* Recomendaciones botánicas ante riesgo inflamatorio

## Arquitectura

La aplicación está estructurada en varios módulos:

components/
Interfaz visual y componentes reutilizables

screens/
Pantallas principales de la aplicación

engine/
Motor metabólico SIM

utils/
Funciones de cálculo metabólico

## Tecnologías utilizadas

* React Native
* Expo
* JavaScript
* AsyncStorage
